# 视觉体验感（单人诊断）

> 若「不想玩下去」主要来自**画面发灰、反馈弱、像原型**，优先看本节，而不是再调数值。

---

## 当前画面为什么容易「没感觉」

| 现象 | 常见原因（本项目） |
|------|-------------------|
| 整体偏暗、发灰 | 后处理 Bloom 曾压得很低；敌人可读性压制会降饱和 |
| 主角不突出 | 敌群压暗 + 相机偏远（0.76 zoom） |
| 击杀无爽感 | 缺少每杀一次的轻量环状反馈 |
| 经验球不显眼 | 豆子在高压场面里不够亮、不够大 |
| 开局像 UI 演示 | 双面板 + 长文案（已可用「记住战备」跳过） |

---

## 已在工程里做的「体验向」默认（v2.2.2+）

- **特效档位**默认 `CINEMATIC`（更亮 Bloom、更高饱和）
- **可读性**默认 `HIGH`（敌人不再大面积洗灰）
- **相机**拉近（0.84 zoom）
- **击杀**：冲击波环 + 轻闪屏（CINEMATIC 每杀；BALANCED 每 2 杀）
- **主角**：更强脚底光晕 + 拖尾
- **经验球**：更亮、更大脉冲
- **天空/后处理**：略提亮、减雾

主菜单若已有「特效档位」，可切换：

- `电影感` → 最亮、反馈最密  
- `均衡` → 中等  
- `竞技` → 最克制（偏淡）

## 前期节奏（摸索窗口）

- 约 **55 秒前**不会弹出升级三选一（经验会攒着，到点再连升）。
- 开局**不再**送半级经验；只撒少量经验球。
- 前 1.5 分钟刷怪更疏，便于试武器和走位。

调参：`assets/config/early_flow_card.json` → `explore_upgrade_min_sec` / `early_spawn_mul_min`

## 你一个人最快拉高「体验感」的 3 步（资产向）

1. **主角立绘**  
   - 在 `Player` 场景设 `external_sprite_path_override` 指向一张**高对比、轮廓清晰**的俯视角角色 PNG（建议 512² 以上）。  
   - 或确认 `res://assets/game_pack/textures/player_chibi.png` 已存在且够醒目。

2. **地面贴图**  
   - 放一张 1024/2048 瓦片图到 `GameDB.TEX_GEN_GROUND` 指向路径（见 `GameDB.gd`），比纯程序灰地面更有「在场感」。

3. **敌人贴图集**  
   - 配置 `TEX_GEN_ENEMY_BASE` / atlas，避免长时间只看色块 multimesh。

改完资源后跑：`python verify_project.py --full`

## 自检清单（30 秒）

- [ ] 画质 **中/高**（低画质会关后处理）  
- [ ] 特效档位 **电影感**  
- [ ] **屏幕闪光** 开启（设置里 `enable_screen_flash`）  
- [ ] 第二局起是否已跳过专精/遗物双面板（`remember_run_loadout`）  
- [ ] 击杀时是否能看到浅色冲击环
